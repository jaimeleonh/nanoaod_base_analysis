dauIdIsoSFRDF

class Tools.Tools.dauIdIso.dauIdIsoSFRDF(**kwargs)[source]

Bases:

Returns the ID and Isolation SF applied to the two leptons with the desired systematics.

Required RDFModules: HHLeptonRDF, eleSFRDF, muSFRDF, tauSFRDF

Parameters:

systs (list of str) – Systematics to be considered. Default: [central, muon_id, muon_iso, ele_iso, tau_vsjet, tau_vse, tau_vsmu].

YAML sintaxis:

codename:
    name: dauIdIsoSFRDF
    path: Tools.Tools.dauIdIso
    parameters:
        isMC: self.dataset.process.isMC
        systs: [central, ...]
