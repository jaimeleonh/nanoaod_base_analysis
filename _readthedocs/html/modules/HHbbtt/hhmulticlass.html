class Tools.Tools.HHMulticlass.HHMulticlassRDF(**kwargs)[source]

Bases:

Returns the HH->bbtt Muticlass DNN outputs.

Lepton and jet systematics (used for pt and mass variables) can be modified using the parameters from JetLepMetSyst.

Parameters:

model_specs (tuple of tuple of str) – versions and models to be used. Syntaxis: ((version, model), ...).

YAML sintaxis:

codename:
    name: HHMulticlassRDF
    path: Tools.Tools.HHMulticlass
    parameters:
        isMC: self.dataset.process.isMC
        year: self.config.year
        model_specs: ((v5 , kl1_c2v1_c31_vbf))
